# BRUTEFORCE
A console application for dictionary-based password recovery of encrypted archives.

The executable file is located on the path:
>bin\Release\SelfContained\Bruteforce.exe

If for some reason it doesn't start, you should install .NET 8.0 Runtime for console apps at the link below:
<https://dotnet.microsoft.com/en-us/download/dotnet/8.0/runtime>



**Features:**
 - Browse files using File Explorer
 - Manual path input
 - Password recovery time measurement

**Supported formats:**
 - *.zip for archives
 - *.txt for dictionaries
